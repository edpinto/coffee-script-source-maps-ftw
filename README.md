# CoffeeScript Source Maps Example Apps
Brought to you by [Eric Wendelin](http://www.eriwen.com)

## NodeJS

#### Prerequisites
```
brew install node
npm install coffee-script -g
```

```
cd nodejs
npm install
node server.js
```

## Rails

#### Prerequisites
* [Install Ruby 2.0](https://coderwall.com/p/tptocq)
* `gem install bundler`

```
cd rails
bundle install
rails s
```
